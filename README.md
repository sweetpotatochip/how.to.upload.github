###깃허브에 파일 업로드 하는법

(1) git init
: git Repository 생성
(2) git status
: (커밋한게 없으면 No commits yet 문구 나옴)
(3) git add * / git add 폴더/파일이름
(4) git commit -m "메시지"
(5) git remote add origin (원격저장소 주소)
: 붙여넣기는 단축키 Shift + Insert
(6) git branch -M main
(6) git push -u origin main
: 모든 변동사항이 레포지토리 github에 업로드됨.
